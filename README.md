# learn-ros2-cavs
An educational repository designed to help students learn how to develop, test, and version ROS 2 packages by building a complete software component from the ground up.

# Project Description
This repository will guide students through the design and development of a **Set-Speed Manager**, a software module responsible for managing the driver's desired cruise control speed. This module will be developed in a ROS2 workspace and be wrapped within a node.

Requirements for this software are provided in an adjacent [REQUIREMENTS](REQUIREMENTS.md) file. The developed module should adhere to the provided requirements and be packaged with test cases that validate the software.

# Pre-requisites
This repository assumes that you have:
- Visual Studio Code
    - Remote Development Extension Pack
    - Devcontainers Extension
- ROS2 Workspace
    - See the [SETUP GUIDE](guides/00_SetupWorkspace.md) if missing

# Getting Started
To get started, follow the provided instructions in the guides subfolder. Make sure to follow the setup guide if you are starting from scratch.
