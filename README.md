# learn-ros2-cavs
This project introduces a requirements-driven approach to software development in a ROS 2 environment. The included guides show the expected workflow to create a validated software component. 

## Project Overview
Students will design a **Set-Speed Manager**, a software module responsible for managing the driver's desired cruise control speed. This module will be developed in a ROS2 workspace and be wrapped within a node.

Requirements for this software are provided in an adjacent [REQUIREMENTS](REQUIREMENTS.md) file. The developed module should adhere to the provided requirements and be packaged with test cases that validate the software.

## Pre-requisites
This repository assumes that you have:
- Visual Studio Code
    - Remote Development Extension Pack
    - Devcontainers Extension
- ROS2 Workspace
    - The [setup guide](guides/00_SetupWorkspace.md) covers how to get this running on your machine

## Getting Started
Follow the provided instructions in the guides subfolder. The setup guide covers cloning this repository, so start there.
