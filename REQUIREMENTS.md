# Set-Speed Manager Requirements

## Functional Requirements
SSM-FR-001: The Set-Speed Manager shall initialize with a set speed of 0 mph.

SSM-FR-002: The Set-Speed Manager shall increase the set speed by 5 mph when incremented.

SSM-FR-003: The Set-Speed Manager shall decrease the set speed by 5 mph when decremented.

SSM-FR-004: The Set-Speed Manager shall limit to the set speed to values between 0 mph and 65mph inclusive.

SSM-FR-005: The Set-Speed Manager shall match the set speed to the current vehicle speed when set and within the limits set in SSM-FR-004.
